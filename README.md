# DataPipelineMiniProject_Springboard
DataPipeline Mini Project for Springboard data engineering career track unit 15.3


Project Description
Load the Ticket Sale CSV file into MYSQL database using Python . Run analytics on top of data stored in MySQL database to find the most popular tickets

Functions Define
get_db_connection() - This function will create the mysql connection load_third_party(connection,filepath) - This function will insert the csv file records into ticket_sales table query_popular_tickets - This function will return the most popular tickets in order of most popular to least popular
